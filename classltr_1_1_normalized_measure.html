Detailed Description

template<class TElement>
class ltr::NormalizedMeasure< TElement >

Normalized measure. Can be used for any approach. Returns lineal normalisation of the given weak measure. Best and worst values of the rasulting measure are given in parameters. Can't normalize infinity measure.
