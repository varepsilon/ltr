Detailed Description

template<typename TElement>
class ltr::utility::TwoClassActualLabelConverter< TElement >

If we have the dataset with less than or equal to 2 different actual labels, then we can use TwoClassActualLabelConverter to convert this actual labels to {-1, 1}. For example, if we have labels 2 and 5, then 2 became -1 and 5 became 1 after converting.
