Detailed Description

template<class TElement>
class ltr::composition::DataSetWeightsUpdater< TElement >

Is a parameter for composition learner. Implements a strategy of updating dataset's element's weights while adding a new scorer to composition scorer. The idea is to increase weights of "hard" elements with bad measure values and decrease weights of "easy" elements with good measure values. Thus next scorer in composition could be learned primary on bad elements and level the limitations of previous scorers
