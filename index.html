About LTR

Learning to Rank (LTR) is an open-source C++ library of algorithms for machine learning problems. Basically LTR can be used for ranking problem but one can use it also for classification and regression problems. To best of our knowledge there is no analogous open-source C++ library for learning to rank.

Philosophy of LTR

From theoretical point of view every algorithms of machine learning can be decomposed to a few simple ideas.
Many of these ideas are similar of even coincident for ranking, classification, regression.
Some examples of decomposition:

  • TreeNet(link) = Gradient Descent + Boosting + Desicion Tree learning
  • SoftRank(link) = Gradient Descent + Neural Net learning + Smoothing Nondifferentiability
  • RankPG(link) = Genetic Programming + Listwise ranking measure optimizing
  • AdaBoost(link) = Boosting + Exponential loss function approximation + Pointwise measure optimizing
  • AdaRank(link) = Boosting + Exponential loss function approximation + Listwise measure optimizing

The main ideas of LTR are easy extensibility and generalizability of source code. This is achieved by wide using of C++ templates and polymorphism.
One can easily add any heuristic to a standard learning algorithm because all of heuristics are functors in LTR. So user can easily change any step of algorithm just defining new heuristic object and insert them into the code of origin algorithm.

User can write code performing learning and testing by himself or can use console application with human-readable configuration file.

Each trained ranker/classifier/regressor can be serialized in C++ code so it can be used in other projects with no dependecies on the LTR library.

Requirements

  • Boost
    LTR assumes that you already have Boost sources and some static libraries in your system. The detailed inatallation manual is here.
  • CMake
    LTR uses the CMake build system so you should install it.

Compiler support

LTR is standard C++98 and should be compatible with any common C++ compiler. LTR is being successfully built with the following compilers:

  • MSVS (Visual Studio) 2008 and newer
  • GCC 4.1 and newer
