Detailed Description

template<class TElement, class TFeatureConverter>
class ltr::BaseFeatureConverterLearner< TElement, TFeatureConverter >

A base class for FeatureConverter learners.

If you want to make your own FeatureConverterLearner you should inherit from BaseFeatureConverterLearner and implement learnImpl().

See also:
FeatureConverterLearner, BaseLearner
